Day 1

JavaScript Variables and Data Types

1. Declaring Variables

In JavaScript, variables can be declared using var, let, or const:

// Using var
var name = 'John'; // Global or function-scoped, can be re-assigned
// Using let
let age = 30; // Block-scoped, can be re-assigned
// Using const
const birthYear = 1990; // Block-scoped, cannot be re-assigned

2. Primitive Data Types

Primitive data types directly hold a single value and are immutable. Examples include:

// String
1 greeting = 'Hello, world!';


// Number
let score = 100;

// Boolean
let isJavaScriptFun = true;

// Undefined
let notAssigned;

// Null
let emptyValue = null;

// Symbol (unique and immutable)
let uniqueId = Symbol('id');

// BigInt (for large integers)
let largeNumber = 1234567890123456789012345678901234567890n;

3. Non-Primitive (Reference) Data Types

Non-primitive data types can hold multiple values and are mutable. Examples include:

// Object
let person = { name: 'Alice', age: 25 };

// Array
let colors = ['red', 'green', 'blue'];

// Function
function sayHello() {
  return 'Hello!';
}

// Date
let currentDate = new Date();

4. Type Coercion and Conversion

JavaScript can automatically convert data types (type coercion) or you can manually convert them (type conversion):

// Type Coercion (automatic)
let result = '5' - 2; // Output: 3 (string '5' coerced to number)
let combined = '5' + 2; // Output: '52' (number 2 coerced to string)

// Type Conversion (manual)
let numberFromString = Number('123'); // Converts string to number: 123
let stringFromNumber = String(123); // Converts number to string: '123'
let booleanFromNumber = Boolean(1); // Converts number to boolean: true
